# Banco de Dados SQL no Azure

---

## 1. Acessar o Portal do Azure
Entre no [portal do Azure](https://portal.azure.com).  
No menu de serviços, procure por **"SQL Database"** (Banco de Dados SQL) e selecione a opção para criar um novo.

---

## 2. Preencher os Dados Básicos
Na tela de configuração, forneça as informações essenciais:

- **Assinatura**: Escolha sua assinatura do Azure.  
- **Grupo de recursos**: Selecione um já existente ou crie um novo para organizar seus recursos.  
- **Região**: Defina a região do servidor.  
- **Nome do banco de dados**: Dê um nome único ao banco.  
- **Nome do servidor**: Crie ou selecione um servidor SQL já existente.

---

## 3. Revisão de Custos e Especificações
O Azure exibirá:

- **Resumo de custos estimados** (armazenamento e computação).  
- **Modelo de uso**: especificações do servidor e tamanho de armazenamento.  

Isso ajuda a ajustar a configuração conforme sua necessidade de desempenho e orçamento.

---

## 4. Revisar e Criar
Na etapa final:

1. Revise todas as configurações.  
2. Concorde com os termos de uso.  
3. Clique em **"Criar"** para provisionar o banco de dados.  

Após alguns minutos, o Azure concluirá a implantação e seu banco de dados estará pronto para uso.
